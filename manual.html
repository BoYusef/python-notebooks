Jupyter notebooks

This course content is provided as Jupyter notebooks that require to be powered by a Jupyter server with Python3 kernel.

Install Anaconda or Miniconda

Anaconda is easy to install but the installer is quite heavy to download because it contains a lot of components. In particular, it is shipped with:

  • Jupyter
  • The Spyder IDE
  • Scipy libraries: Numpy, Pandas, etc.

Miniconda is a light version of Anaconda.

  • Miniconda installer for Linux .
  • Anaconda installer for Linux

In your terminal window, run:

  • Miniconda: 
bash Miniconda3-latest-Linux-x86_64.sh
  • Anaconda: 
bash Anaconda-latest-Linux-x86_64.sh

Follow the prompts on the installer screens. If you are unsure about any setting, accept the defaults. You can change them later. To make the changes take effect, close and then re-open your terminal window. Test your installation by using your terminal window or Anaconda Prompt, run the command conda list . A list of installed packages appears if it has been installed correctly.

Finalize installation with conda from the project root directory, type: 

git clone https://github.com/pnavaro/python-notebooks
cd python-notebooks
conda env update -f environment.yml

Start a Jupyter server

  • Either from Anaconda graphical interface
  • or from the command line interface from the project root directory: 
jupyter-notebook
